ML App for Detecting BankNotes From Fraud

This app uses a trained machine learning model to predict whether a banknote is original or fake.

How to use this app?

Enter the four numerical features extracted from a banknote image:

  • Variance: spread of image pixel values
  • Skewness: asymmetry of the image data
  • Curtosis: shape/sharpness of the distribution
  • Entropy: randomness or noise in the image

After entering the values, click Predict.
